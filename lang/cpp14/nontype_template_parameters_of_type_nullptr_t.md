# nullptr_t型の定数式を非型テンプレートパラメータとすることを許可
* cpp14[meta cpp]

<!-- start lang caution -->

のちのC++規格でさらに変更される場合があるため[関連項目](#relative-page)を参照してください。

<!-- last lang caution -->

## 概要
[`nullptr_t`](/reference/cstddef/nullptr_t.md)型の定数式を非型テンプレートパラメータとすることが許可された。


## 例
```cpp example
#include <cstddef>

template <std::nullptr_t P>
struct X {};

int main()
{
  X<nullptr>();
}
```
