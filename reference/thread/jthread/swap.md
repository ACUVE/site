# swap
* thread[meta header]

```cpp
void swap(jthread& x) noexcept;
```

## 概要
別の`jthread`と交換する。


## 効果
`*this`と`x`を入れ替える。


## 例外
送出しない。


## 例
```cpp example
#include <thread>

int main()
{
  std::jthread jt1([]{ /*...*/ });
  std::jthread jt2;

  jt1.swap(jt2);

  jt2.join();

  return 0;
}
```
