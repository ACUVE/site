# MB_LEN_MAX
* climits[meta header]
* macro[meta id-type]

```cpp
#define MB_LEN_MAX implementation-defined
```

## 概要
サポートされているすべてのロケールのマルチバイト文字のうち、最大のバイト数を表す。`#if` などのプリプロセッサディレクティブで使用できる。  
具体的な値は実装依存であるが、`1` 以上であることが規格で定められている。

## 例
```cpp example
#include <climits>
#include <iostream>

int main()
{
  std::cout << MB_LEN_MAX << '\n';
}
```


### 出力例
```
16
```
